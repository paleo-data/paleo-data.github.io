This page contains resources to help collections professionals with the digitization and data management of paleontology collections. Feel free to add any content to these resources or suggest a resource that should be added to the list.

<i class='fa fa-arrow-right fa-lg'></i> ___[Draft interpretation of Darwin Core terms for the paleo context](https://docs.google.com/spreadsheets/d/1RoZXAoj5LNFjokMUREKLcpepgX7LugpVWY9PcNePscA/edit?usp=sharing)___. The aim of this document is to collect information about best practices on using Darwin Core terms for paleo collections.

<i class='fa fa-arrow-right fa-lg'></i> ___[Review of Best Practices for Recording Paleo Taxonomy](https://docs.google.com/document/d/1hI15A97VKMdZ3jM7_JjMTcIkrSbe7tyO1GczKybYeYc/edit?usp=sharing)___. This document outlines questions, prompts, and resources to guide review of best practices for recording paleo taxonomy in local, institutional collections information systems. The goal will be to create a guideline for the paleo community with suggested best practices that encourage consistency in our data, with an aim to improve quality in downstream use and data aggregation

<i class='fa fa-arrow-right fa-lg'></i> ___[Review of Best Practices for Sharing Paleo Taxonomy](https://docs.google.com/document/d/1DE28Bx-Q3HDGDmDvvZPxKC67Nq6IuOvc-LJC2kgv4jA/edit?usp=sharing)___. This document outlines questions, prompts, and resources to guide review of best practices for sharing paleo taxonomy, as it maps to global standards like DwC and addressing ingest processes by aggregators. The goal will be to create a guideline for the paleo community with suggested best practices that encourage consistency in our data, with an aim to improve quality in downstream use and data aggregation.

<i class='fa fa-arrow-right fa-lg'></i> ___[Taxonomic Authorities in Use by Paleo Collections](https://docs.google.com/document/d/12NYKco_4EUIx_cttvaqf1oULSFuJRu5h4hgbL3qqvxU/edit?usp=sharing)___. This document is an annotated list of what taxonomic authorites are typically used for various subsections of paleontology, including notes on digital accessibility of each authority.

<i class='fa fa-arrow-right fa-lg'></i> ___[Review of Geologic Context Data: Terms, Vocabulary, Best Practices, and Aggregator Handling](https://docs.google.com/document/d/1MHTpcbJxeCqhgfmVCooe6u3uE4kTCwOhBvk8j2jB4VA/edit?usp=sharing)___. This document is a working draft of our efforts to improve how geologic context data are mobilized and made discoverable.

<i class='fa fa-arrow-right fa-lg'></i> ___[Paleo Identifiers Guide](https://docs.google.com/spreadsheets/d/1nllxirFg-JJ0zQ7259c1612oKgXVsAx0bYHOE_45K8Y/edit?usp=sharing)___. This spreadsheet is an aggregated list of all the identifiers in use by or potentially useful to members of the paleo collections community.

<i class='fa fa-arrow-right fa-lg'></i> ___[Sensitive Localities](https://docs.google.com/document/d/1_ujKKB8We7KUqwL_SoByTEq6bsgg9FGybwOqKjAi0Ks/edit?usp=sharing)___. This document is a draft guideline describing how to treat sensitive locality data for fossil specimens, especially in the context of data mobilization to aggregators.
